# Repository öffentlich machen

## Schritt-für-Schritt Anleitung

1. **Gehe zu Repository Settings:**
   - Öffne: https://github.com/9qdbds6xqc-png/trafosanf-remake/settings

2. **Scrolle nach unten:**
   - Gehe zum Bereich **"Danger Zone"** (ganz unten, roter Bereich)

3. **Klicke auf "Change visibility"**

4. **Wähle "Make public"**

5. **Bestätige:**
   - Gib ein: `9qdbds6xqc-png/trafosanf-remake`
   - Klicke **"I understand, change repository visibility"**

## Nach dem öffentlich machen:

1. Warte 1-2 Minuten
2. Gehe zu: https://github.com/9qdbds6xqc-png/trafosanf-remake/settings/pages
3. Jetzt solltest du das "Pages" Menü sehen können
4. Unter "Source" wähle: **"GitHub Actions"**
5. Klicke "Save"

## Alternative Route:

Falls das Pages-Menü immer noch nicht erscheint:
1. Gehe zu: https://github.com/9qdbds6xqc-png/trafosanf-remake/actions
2. Klicke auf "Deploy to GitHub Pages"
3. Klicke "Run workflow" → "Run workflow"
4. Das aktiviert GitHub Pages automatisch
